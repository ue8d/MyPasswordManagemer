# passwordManagement

パスワードの作成、管理目的に作成しているソフトです。

## ダウンロード - v0.1
https://ue8d.com/Release/passwordManagement_v0.1.exe

## 使い方
1. 起動後に画面の指示に従いパスワードの保存先を選んで、必要項目を入力して登録をクリック

1. DBを選択した場合はアプリを再起動
    1. 再起動後は先ほど登録したマスターパスワードを入力し、ログインをクリック

1. オプション項目を入力・選択し、パスワード作成をクリック

1. サイト名を入力して保存をクリック
